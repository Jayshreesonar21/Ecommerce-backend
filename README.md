# Node version

Node v20+

# Run project:

> Step 1:

- Create DATABASE in "pgAdmin"

> Step 2:

- Create .env file

> Step 3: Run migration using following command:

- npm run migration:run

> Step 4: Run Seeder for Role & Admin creation:

- npm run seed

> Step 5: Import Postman collection file from 'Ecommerce Backend.postman_collection.json'

> Ste 6: Run project using following command:

- npm run dev
